快速开始：Hugo + GitHub Pages

推荐新手选择 Hugo（极速构建）配合 GitHub Pages（免费托管），十分钟即可上线

安装 Hugo

根据你的操作系统安装 Hugo 命令行工具

# macOS
brew install hugo

# Windows
winget install Hugo.Hugo.Extended

创建新站点

使用 Hugo 命令初始化博客项目

hugo new site my-blog
cd my-blog
git init

添加主题

从 Hugo 官方主题库选择一个喜欢的主题，以子模块方式添加

git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
echo "theme = 'ananke'" >> hugo.toml

创建内容

使用 Markdown 格式撰写你的第一篇文章

hugo new content posts/hello.md
# 编辑 content/posts/hello.md 添加内容

部署到 GitHub Pages

使用 GitHub Actions 自动构建和部署

# 推送到 GitHub 后，在仓库 Settings > Pages 中配置
# 选择 GitHub Actions 作为 Source
# Hugo 官方提供现成 workflow 模板
